Evaluation of Web Graphics

Microsoft

The Microsoft site uses graphics effectively in how it uses good quality graphics that are simple and there are just the right amount of images.

  1. Referring to the section in the handout titled Purpose of graphics in web design, what do you feel is the purpose of the graphics on the page you selected?
    • The purpose is to show different Microsoft products, while keeping the design simple so everyone can use it.


  2. Referring to the section in the handout titled Guidelines for effective use of web graphics, how well does the page you selected meet these guidelines? Which guidelines does it do especially well on, if any? Which guidelines does it fail to meet?
    • The Microsoft site does follow the guidelines very well. There are not too many images, and they are small enough to convey information, but not too big to slow down page loading. Not every image has an "alt" specified, however.
